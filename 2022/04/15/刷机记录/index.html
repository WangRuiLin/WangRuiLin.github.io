刷机记录

发表于 2022-04-15 分类于 安卓逆向 本文字数： 1.7k 阅读时长 ≈ 4 分钟

本文主要记录Nexus 5、Pixel XL以及Redmi Note 9 (4G)的刷机过程。

Nexus 5

准备

  1. 下载Nexus 5系统镜像。地址：https://developers.google.com/android/images#hammerhead
  2. 下载Twrp刷机工具。下载地址：https://twrp.me/lg/lgnexus5.html
  3. 下载Magisk。地址：https://github.com/topjohnwu/Magisk/releases
  4. 下载Xposed。地址：https://dl-xda.xposed.info/framework/。

刷机

  1. 执行adb reboot bootloader， 重启手机至bootloader。

  2. 如果直接执行系统镜像中的./flash-all.sh，手机会无限重启。出现此问题，建议手动清除各个分区，并且手动刷各个分区image。自己写一个shell脚本，命名为./flash-all-new.sh，并执行。

  3. 等待手机重启，刷机至此完成。

  4. 打开开发者模式以及开启USB调试

刷recovery

  1. 执行adb reboot bootloader，重启手机至bootloader。
  2. 执行fastboot flash recovery twrp-3.4.0-0-hammerhead.img。
  3. 按上下音量键，选择recovery。按电源键确认。

刷magisk

  1. adb push Magisk-v23.0.zip /sdcard/
  2. reboot system

刷Xposed

  1. adb push xposed-v89-sdk23-arm.zip /sdcard/
  2. reboot system

Redmi Note 9(4G)

准备

  1. 下载并解压Fastboot线刷包。下载地址：https://xiaomirom.com/rom/redmi-note-9-5g-cannon-china-fastboot-recovery-rom/
  2. 下载并安装MiFlash刷机工具。注意，MiFlash仅支持Windows平台。相关介绍及下载地址：https://xiaomirom.com/download-xiaomi-flash-tool-miflash/

刷机

  1. 点击MiFlash右侧->加载设备。
  2. 点击MiFlash左侧->选择，选择准备1中解压好的系统镜像路径。
  3. 点击MiFlash右侧->刷机。静静等待几分钟，系统启动。
  4. 打开开发者模式。设置->我的设备->全部参数->MIUI版本，连续点击数次。
  5. 打开USB调试。设置->更多设置->开发者选项，打开USB调试。

注意：

  1. 不建议在fastboot模式下直接执行./flash_all.sh。
  2. 建议执行./flash_all_lock.bat。亲测./flash_all.bat会报错，执行失败。

踩过的坑

  1. Twrp并没有支持Redmi Note 9，找了好久，其实用MiFlash就好了，简单方便。
